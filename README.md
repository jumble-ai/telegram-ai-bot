# tg-agent

🤖 Telegram-бот на aiogram 3.27+, который общается через OpenRouter, помнит
последние 10 сообщений в SQLite, рисует изображения и ищет информацию в сети.
Также поддерживает голосовые сообщения, TTS и донаты через Crypto Pay.
Telegram voice конвертируется из OGG/Opus в WAV внутри Python через PyAV.

## Запуск

1. Скопируйте `.env.example` в `.env`.
2. Укажите `BOT_TOKEN`, полученный у BotFather.
3. Укажите `OPENROUTER_API_KEY`, полученный в OpenRouter.
4. При необходимости поменяйте `OPENROUTER_MODEL`.
5. При необходимости поменяйте `OPENROUTER_IMAGE_MODEL`.
6. При необходимости поменяйте `OPENROUTER_STT_MODEL`.
7. При необходимости поменяйте `OPENROUTER_TTS_MODEL`.
8. При необходимости поменяйте `OPENROUTER_TTS_VOICE`.
9. Укажите `TAVILY_API_KEY`, полученный в Tavily.
10. Укажите `CRYPTO_PAY_API_TOKEN`, полученный в Crypto Pay.
11. Укажите `GOOGLE_SERVICE_ACCOUNT_FILE` и расшарьте календарь на email
service account.
12. Укажите в `GOOGLE_CALENDAR_ID` ID календаря, расшаренного на service
account. Для личного основного календаря это обычно email Google-аккаунта;
не используйте `primary`, потому что для service account это его отдельный
календарь.
13. При необходимости поменяйте `GOOGLE_CALENDAR_TIMEZONE`.
14. При необходимости поменяйте `DATABASE_PATH`.
15. Установите зависимости и запустите бота:

```bash
uv sync
uv run tg-agent
```

## Запуск через Docker Compose

На VPS с Ubuntu установите Docker и плагин Compose, положите рядом с проектом
`.env` и JSON-файл service account, затем запустите:

```bash
docker compose up -d --build
```

Логи:

```bash
docker compose logs -f bot
```

SQLite хранится в примонтированной папке `data/`, поэтому история переживает
перезапуск контейнера.

## Использование

💬 Обычный текст отправляется в чат-модель с памятью последних 10 сообщений.

🎨 Для генерации изображения:

```text
/image кот в космосе в стиле ретрофутуризма
```

🔎 Для поиска в сети:

```text
/search последние новости aiogram 3.27
```

🎙️ Голосовое сообщение бот конвертирует в WAV, распознает и отвечает текстом.
Также поддерживаются аудиофайлы `.mp3` и `.wav`.

🔊 Для озвучки текста:

```text
/tts Привет! Это голосовой ответ.
```

💝 Для доната:

```text
/donate10
/donate100
```

🧾 Для создания счета через Crypto Pay напишите обычным текстом:

```text
Создай счет на 15$
```

Бот распознает намерение через ИИ, создаст USD-счет с оплатой в USDT, TON или
BTC и вернет ссылку с кнопкой «Поделиться».

📅 Для создания события в Google Календаре напишите обычным текстом:

```text
Добавь встречу завтра с 13:00 на полчаса
```

Бот распознает намерение через ИИ, создаст событие и вернет кнопку
«Посмотреть событие».
